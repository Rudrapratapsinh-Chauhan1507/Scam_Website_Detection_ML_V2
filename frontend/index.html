⚙️ ML Pipeline
Exact logic from final_predict_model.ipynb
🔄 Pipeline Steps
1

URL Normalization

Prepend https:// if missing. Scraper follows redirects and returns final_url.

2

WebsiteScraper.scrape()

DNS resolve → SSL-verified fetch (3 retries, HTTP fallback). Extracts title, text, HTML via BeautifulSoup.

3

Feature Extraction (3 groups)

All on final_url: URLFeatureExtractor + DomainFeatureExtractor + ContentFeatureExtractor(title, text, html).

4

build_vector(features)

DataFrame → numeric coerce → fill missing from feature_columns.pkl → reorder to training column order.

5

model.predict_proba(X)[0][1]

Random Forest outputs base_proba. No scaler needed.

6

Rule Engine

Accumulate scam_score + legit_boost from 13 domain signals. Final = clamp(0.85×ML + scam − legit, 0, 1).

7

Decision

Shortened → SUSPICIOUS (force ≥0.65). Trusted → cap 0.45. Else: ≥0.80=SCAM, ≥0.60=SUSPICIOUS, else LEGIT.

⚖️ Rule Engine Logic
TRUST (legit_boost) .edu/.gov/.ac./.org/.mx +0.40 domain_age > 1000 days +0.05 SCAM (scam_score) is_shortened +0.25 wpenginepowered/ herokuapp/firebaseapp/ github.io/netlify/ vercel/000webhost +0.30 num_hyphen >= 5 +0.15 subdomains >= 3 +0.15 suspicious_tld +0.20 weebly/blogspot/ wixsite/wordpress +0.25 no https +0.05 url_length > 80 +0.05 path: login/verify/ account/update/secure/ wp-admin +0.20 scam_kw_density > 0.05 +0.15 has_form (untrusted) +0.05 has_iframe (untrusted) +0.05 FORMULA p = clamp(0.85*ML+scam-legit, 0,1) trusted: p = min(p, 0.45) shortened: label=SUSP, p=max(p,0.65)
